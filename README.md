# 🎮 Games Arena

A modern **front-end gaming landing page** built with **HTML5, CSS3, and Bootstrap 5**.

The project focuses on creating a visually engaging UI inspired by gaming platforms while maintaining full responsiveness and clean structure across all devices.

---

## 🌐 Live Demo
🔗 [View Project](https://yarahossam1.github.io/Games-Arena/)

---

## ✨ Features

-  Fully responsive design (mobile, tablet, desktop)
-  Dark gaming-themed UI
-  Responsive navigation bar for smooth browsing
-  Carousel section for featured games
-  Team showcase with hover effects
-  Smooth animations & transitions
-  Styled contact form
-  Clean layout using Bootstrap Grid system

---

## 🧰 Built With

- HTML5 – semantic structure
- CSS3 – styling, animations, and effects
- Bootstrap 5 – responsive layout & components
- Flexbox – alignment & positioning
- Media Queries – responsiveness control
- CSS Variables – consistent design system
